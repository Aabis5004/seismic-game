- 15 levels across 3 chapters

- Global leaderboards
- Works on desktop & mobile

## Quick Start

```bash
npm install
npm start
```

Then open: http://localhost:3000

## Deploy to VPS

1. Clone this repo
2. Run `npm install`
3. Run `pm2 start server.js --name seismic-game`
4. Access via your VPS IP on port 3000
